# 登录与仪表盘

## 功能概述

后台管理系统的入口。登录后进入 Dashboard 首页，展示站点统计数据和快捷导航。

## 操作步骤

### 1. 登录

1. 打开浏览器，访问后台地址（开发环境：`http://localhost:5173`）
2. 输入邮箱和密码
3. 点击「登录」按钮
4. 登录成功后自动跳转到 Dashboard

> 首次部署的默认超管账号：`admin@example.com`，密码在 `.env` 中配置（默认 `admin123`）

### 2. Dashboard 概览

登录后进入 Dashboard，页面包含：

- **欢迎语**：显示当前登录用户名
- **统计卡片**：文章总数、标签总数、分类总数、今日访问（当前为占位，Phase 3 接入真实数据）
- **快捷导航**：通过左侧侧边栏访问各管理模块

### 3. 退出登录

1. 点击页面右上角用户名
2. 选择「退出登录」
3. 系统自动清除 JWT Token，跳转到登录页

## 注意事项

- **密码要求**：至少 6 位字符
- **Token 过期**：Access Token 有效期 2 小时，过期后会自动尝试刷新；Refresh Token 有效期 7 天
- **登录失败**：连续错误不会锁定账号，但请确认邮箱和密码正确
- **权限不足**：如果登录后看不到某些菜单，说明当前账号没有对应权限，请联系超管分配角色

## 权限要求

- 登录：无需权限（公开路由）
- Dashboard：需要已登录（任何已认证用户）
