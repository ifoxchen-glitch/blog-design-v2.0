# 标签与分类管理

## 功能概述

标签和分类用于组织文章内容。标签更灵活（可跨分类），分类更结构化（有层级关系）。

## 操作步骤

### 1. 标签管理

**进入**：侧边栏 → 博客管理 → 标签

#### 新建标签

1. 点击「新建标签」
2. 填写：
   - 名称：如「Vue3」
   - Slug：URL 标识，留空自动生成
3. 保存

#### 编辑标签

1. 点击「编辑」
2. 修改名称或 Slug
3. 保存后所有关联文章自动更新

#### 删除标签

1. 点击「删除」
2. 确认删除
3. 删除后，关联文章不再显示该标签（文章本身不受影响）

### 2. 分类管理

**进入**：侧边栏 → 博客管理 → 分类

#### 新建分类

1. 点击「新建分类」
2. 填写：
   - 名称：如「前端开发」
   - Slug：URL 标识
3. 保存

#### 编辑/删除

操作与标签相同。删除分类后，关联文章不再归属该分类。

## 注意事项

- **Slug 规则**：只能包含小写字母、数字、连字符（如 `front-end`）
- **名称唯一**：同一标签/分类名称不能重复
- **文章关联**：在文章编辑页选择标签和分类，不是在标签/分类页管理关联
- **前台展示**：标签云和分类列表由前台自动渲染

## 权限要求

| 操作 | 所需权限 |
|---|---|
| 查看标签/分类 | `post:list` |
| 新建 | `post:create` |
| 编辑 | `post:update` |
| 删除 | `post:delete` |
