# 文章管理

## 功能概述

文章是博客的核心内容。支持创建草稿、编辑内容、发布上线、下架回退、删除等完整生命周期管理。

## 操作步骤

### 1. 查看文章列表

**进入**：侧边栏 → 博客管理 → 文章

列表展示：
- ID、标题、摘要、封面图、状态（草稿/已发布）
- 发布时间、更新时间
- 标签和分类（鼠标悬停查看）

**筛选**：
- 顶部搜索框：按标题关键词搜索
- 状态筛选：全部 / 草稿 / 已发布

### 2. 新建文章

1. 点击「新建文章」按钮
2. 进入编辑页，填写：
   - **标题**：文章标题（必填）
   - **Slug**：URL 友好标识，留空会自动根据标题生成
   - **摘要**：可选，不填会自动截取正文前 200 字
   - **封面图**：可选，支持粘贴图片 URL 或使用媒体库上传
   - **正文**：Markdown 格式，使用 Vditor 编辑器
   - **标签**：输入或选择已有标签（支持多选）
   - **分类**：选择文章分类（支持多选）
3. 点击「保存草稿」或「保存并发布」

### 3. 编辑文章

1. 在列表中点击「编辑」
2. 修改内容后保存
3. 已发布的文章编辑后保持「已发布」状态

### 4. 发布 / 下架

- **发布**：草稿状态的文章，点击「发布」按钮立即上线
- **下架**：已发布的文章，点击「下架」按钮变回草稿，前台不再展示

### 5. 删除文章

1. 点击「删除」按钮
2. 确认弹窗中确认删除
3. 删除后不可恢复

### 6. 批量操作

1. 勾选列表左侧复选框
2. 选择批量操作：批量删除

## Markdown 编辑器

正文使用 Vditor 编辑器，支持：
- 所见即所得模式
- Markdown 源码模式
- 图片拖拽上传（自动上传到媒体库）
- 代码块、表格、任务列表等语法

## 注意事项

- **Slug 唯一性**：同一 Slug 的文章会覆盖，系统会自动去重
- **标签自动创建**：输入不存在的标签名，保存时会自动创建
- **分类必须存在**：只能选择已有分类，需先在分类管理页创建
- **封面图尺寸**：建议 1200×630 像素，支持 JPG/PNG/WebP
- **前台生效**：发布后访问 `http://localhost:8787/post/<slug>` 查看

## 权限要求

| 操作 | 所需权限 |
|---|---|
| 查看文章列表 | `post:list` |
| 新建文章 | `post:create` |
| 编辑文章 | `post:update` |
| 删除文章 | `post:delete` |
| 发布/下架 | `post:update` |
