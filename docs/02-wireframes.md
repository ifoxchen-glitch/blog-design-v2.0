# 低保真线框（模块级）

以下为 **区块自上而下顺序**，与 `../index.html`、`archive.html`、`post.html`、`about.html` 结构一致，便于对照。

## 首页

```
+--------------------------------------------------+
| [Logo]  首页  归档  分类  标签  关于      [搜索?] |
+--------------------------------------------------+
| HERO: 站点一句话 + 副标题                         |
| [可选主视觉 / 插图占位]                           |
+--------------------------------------------------+
| 最新文章                                          |
| +------+  标题                                    |
| | 封面 |  摘要两行…                               |
| +------+  日期 · 分类                             |
| +------+  标题                                    |
| |      |  …                                       |
+--------------------------------------------------+
| 页脚：版权 · RSS · 社交图标                       |
+--------------------------------------------------+
```

## 归档 / 列表（分类、标签同布局）

```
+--------------------------------------------------+
| [Logo]  导航…                          [菜单≡]   |
+--------------------------------------------------+
| 页面标题：「全部文章」/「分类：xxx」              |
| [可选：年份或标签 chips 筛选]                     |
+--------------------------------------------------+
| [卡片] [卡片] [卡片]   ← 桌面可 2–3 列网格       |
| [卡片] [卡片] [卡片]   ← 移动单列                |
+--------------------------------------------------+
| [上一页]  1 2 3  [下一页]                        |
+--------------------------------------------------+
| 页脚                                              |
+--------------------------------------------------+
```

## 单篇（长文）

```
+--------------------------------------------------+

|                        | 图片 + caption            |
+--------------------------------------------------+
| 相关阅读 / 上一篇 · 下一篇                        |
| [可选 Newsletter 条]                              |
+--------------------------------------------------+
| 页脚                                              |
+--------------------------------------------------+
```

## 移动端导航

- 顶栏右侧：**菜单控件**（`details/summary` 实现，无 JS）。
- 展开：**垂直链接列表** + 可选搜索占位。

详见 `../index.html` 中 `<details class="nav-mobile">` 与 `css/components.css`。
